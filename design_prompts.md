# Prompts para Diseño UI (Google Stitch / AI Tools)

Copia y pega estos prompts en Google Stitch o tu herramienta de generación de UI preferida.

## 1. Prompt General (Estilo y Estructura)
> "Create a comprehensive school attendance management web application called 'EduBeta'. The design should be modern, minimalist, and highly responsive, using a 'clean slate' aesthetic with a colour palette of white, slate grays, and primary indigo/blue accents. Use rounded corners (radius-lg) for cards and inputs, and use the Inter font family. The layout should feature a collapsible dark sidebar on the left for navigation and a light gray background for the main content area."

## 2. Prompt para Dashboard Admin
> "Design an Admin Dashboard view for EduBeta. Needs a top header with user profile. The main content area should have a grid of summary cards (Stats for 'Total Students', 'Attendance Rate', 'Active Teachers'). Below the stats, include a fast-access table showing recent attendance reports with status badges (Present=Green, Absent=Red, Late=Yellow). Include a Floating Action Button (FAB) or prominent 'Add New' button at the top right."

## 3. Prompt para Toma de Asistencia (Vista Profesor)
> "Create a 'Take Attendance' interface for teachers. Display a header with the current Class Name and Date. The main body should be a list of students. Each student row must have their Avatar, Name, and a segmented control or toggle group to select status: 'Present', 'Absent', 'Late'. At the bottom, place a sticky footer with a large primary 'Save Attendance' button."

## 4. Prompt para Gestión de Estudiantes (Lista)
> "Design a Data Table view for managing Students. Includes a search bar and filter dropdowns (by Grade) at the top. The table columns should be: Student Name, Grade, Parent Name, and Actions (Edit/Delete icons). Use zebra striping or hover effects for rows. Add pagination controls at the bottom."
