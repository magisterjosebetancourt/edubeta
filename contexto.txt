# CONTEXTO DEL PROYECTO: EduBeta

## Características del Proyecto
EduBeta es una plataforma de gestión académica integral diseñada para instituciones educativas. Permite gestionar estudiantes, docentes, inasistencias, grados, asignaturas y caracterización socioeconómica.

## Aplicación
- **Frontend**: React (Vite) + Tailwind CSS + Lucide Icons + Shadcn UI.
- **Backend**: Firebase (Firestore para base de datos, Authentication para usuarios).
- **Branding**: Marca "BETASOFT" con estética premium (HSL tailored colors, micro-animaciones).

## Avances y Resumen de Planes
- **Migración a Firebase**: Se completó la transición desde Supabase a Firebase Firestore y Auth. Todas las vistas principales han sido migradas.
- **Implementación de Branding**: Se estableció la identidad visual inicial "BETASOFT" en todo el dashboard.
- **Corrección de Errores**: Se resolvieron problemas de inicialización de hooks en la vista de docentes y errores de índices en Firestore mediante optimización en memoria.

## Decisiones Tomadas
1. **Firebase**: Elegido por su facilidad de integración y escalabilidad para tiempo real.
2. **Ordenamiento en Memoria**: Decidido para evitar la complejidad de crear índices compuestos en Firebase cada vez que se requiera un filtro/orden simple, mejorando la agilidad del desarrollo.
3. **Diseño Mobile-First**: Estructura de navegación adaptable optimizada para dispositivos móviles (MobileMenuPage).
